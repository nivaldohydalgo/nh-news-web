DISCLAIMER
Este site não pretende ser um veículo de noticias.
Trata-se somente de um experimento da utilização de tecnologias modernas de desenvolvimento de aplicativos, como Containers e Cloud Computing.
TECNOLOGIAS UTILIZADAS
HTML, CSS, Javascript, SQL, Containers e Cloud Computing.
IMPLEMENTAÇÃO
FRONTEND
Desenvolvido com Angular, utilizando Angular Material, e hospedado no GIT Pages.
BACKEND
Desenvolvido com NodeJS, utilizando Express, executando em um container Docker e hospedado no Google Cloud Run.
BANCO DE DADOS
Dados armazenados utilizando o PostgreSQL, executando em uma Maquina Virtual com Ubuntu no Google Cloud.
BANCO DE IMAGENS
As imagens utilizadas estão armazenadas em um Bucket da AWS S3.
